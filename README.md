# Azeroth Agents

Dossier classé confidentiel — SI:67.

Dans l’ombre des capitales, loin des champs de bataille et des tavernes bruyantes, une cellule discrète recrute les esprits les plus affûtés d’Azeroth. Leur mission : identifier les agents infiltrés, éviter les civils, démasquer les pièges ennemis… et surtout, ne jamais croiser la route de l’assassin.

**Azeroth Agents** est un addon de mini-jeu sur World of Warcraft inspiré des jeux de déduction par équipes. Formez deux camps, désignez vos maîtres-espions, donnez des indices subtils et tentez de retrouver tous vos agents avant l’équipe adverse.

La victoire appartient rarement aux plus bruyants.  
Elle revient à ceux qui savent lire entre les lignes.
